About

I'm a Computer Science student at the University of Arkansas (graduating Spring 2027), focused on full-stack web development. I like owning a project end-to-end — from database schema and API design through real-time front-end features and deployment.

Most of my work lives in the React, TypeScript, and Node.js world. I care about getting the hard parts right where they actually belong: enforcing data integrity at the database layer, syncing real-time state to every client, and designing auth that balances security against friction — not just patching things in the UI.

Outside of building, I'm an active member of the ACM chapter on campus, where I work through weekly algorithmic challenges in Python and C++. I'm currently looking for software engineering internship opportunities.
